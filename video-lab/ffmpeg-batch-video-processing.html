FFmpeg 是什么

FFmpeg 是一个开源的命令行视频处理工具，几乎支持所有视频格式的转码、剪辑、合并、滤镜处理。

最常用的几条命令

视频转码

ffmpeg -i input.mp4 -c:v libx264 -crf 23 output.mp4

裁剪视频（保留指定时间段）

ffmpeg -i input.mp4 -ss 00:01:30 -t 60 -c copy output.mp4

批量处理脚本

for f in *.mp4; do
  ffmpeg -i "$f" -vf "scale=1920:1080" "resized/$f"
done

实际应用场景

  • 航拍素材批量转 ProRes 格式给 DaVinci 用
  • 多个短片段合并成一个完整视频
  • 提取视频里的音频转 MP3

命令行工具学习曲线陡，但学会了效率提升非常明显。
