01分发策略（主理人·安拓）

定位：综合科技随笔（长文 + 视频双形态）· 目标：专业影响力 + 个人品牌 · 节奏：每周 1–2 更 · 覆盖 9 平台：微信生态（公众号 / 视频号）、小红书、B站、抖音、YouTube、TikTok、Instagram、X、LinkedIn。制定：全域内容分发专家团。

核心骨架：一次生产、多端分发

每周产出 1 篇主稿长文（硬指标）+ 每 2 周 1 支中视频，再按平台做「比例 + 调性 + 标签」三件套改造，严禁原样搬运（各平台有查重去重机制）。

平台优先级分层（先打透核心层，再外扩）

  • 🔴 核心层（做深）：微信公众号、B站、小红书
  • 🟡 增长层：视频号、抖音、YouTube
  • 🟢 出海层：X、LinkedIn、TikTok、Instagram

预期心智

  • 国内开发者 = 微信 + B站 + 小红书 三件套
  • 海外开发者 = X + LinkedIn + YouTube + TikTok
  • 长尾资产 = appin.site 技艺录（网站搜索沉淀）

批量编排 SOP（排期日集中处理）

  1. 排期日（建议周日 21:00 或周一 09:00）：确认主稿主题与形态，调取各平台适配稿。
  2. 录入定时，区分自动 / 手动：自动（排期日一次性设）— 公众号、B站、YouTube、TikTok、Instagram、LinkedIn、X；手动（到点发）— 小红书、视频号、抖音。
  3. 海外统一入口：所有海外平台 Bio / 置顶放 Linktree（聚合公众号 / Newsletter / B站），避免重复导流文案。
  4. 发布日只做确认 + 互动：自动项核查是否发出；手动项到点发，视频号首发 24h 朋友圈 + 群互动（冷启动关键）。

跨时区错峰策略

国内黄金 12–13、19–23 点与北美 08–10、17–19 ET 天然错开 12h。

坤哥睡前 1 小时（BJT 20:00–22:00）= 北美起床第 1 小时（ET 08:00–10:00）：周一主稿周二同时驱动公众号（国内晚）+ LinkedIn / X（北美早），一次生产喂饱两时区。

夏令时 EDT 按 BJT = ET + 12h；冬令时 EST 整体 +1h，排期日前强制核对一次偏移。

发布频率与节奏红线

  • 硬指标：主稿 1 篇 / 周（断更不超 1 周）；中视频 1 支 / 双周（可延至 3 周，别断超 1 月）。
  • 可日更观点：X（3–5 条 / 周，重质不硬凑）。
  • 慢养型（质量 > 数量）：B站、小红书、YouTube。
  • 社交驱动：视频号（求冷启动互动）、抖音（切片为主）。
  • 低频精准：LinkedIn（1–2 条 / 周）、Instagram / TikTok（双周切片）。
  • 断更保护：至少发 1 条 X 短评 + 重发 1 篇小红书 evergreen 旧文，维持算法信任。

发布检查清单

  • [ ] 主稿定稿，各平台适配稿到位
  • [ ] 公众号 OAuth2 定时（周一 20:30）已设
  • [ ] 海外 5 平台定时已设（YT 六 03:00 / LinkedIn 二 & 四 20:30 / X 三发 / TikTok 四 08:00 / IG 三 23:00）
  • [ ] B站 Webhook 定时（视频周 五 19:00）已设
  • [ ] 小红书 / 视频号 / 抖音 草稿已存、手机提醒已设
  • [ ] 发布日：自动项核查、手动项到点发、视频号首发 24h 朋友圈 + 群互动、评论区互动 30min
  • [ ] 周末：记录数据、标记优质内容（下周公号 / 小红书 evergreen 重发）、预填下周排期

异常处理预案

  • 公众号 OAuth2 失效 → 当天 20:30 前手动群发，检查 token。
  • B站 Webhook 失败 → 后台手动上传定时。
  • 海外定时未发 → 查 Later / Meta / Buffer 登录态，手动补最近窗口。
  • 抖音 / 小红书 / 视频号忘发 → 当日 20:00 前补，错过则次日补。
  • 主稿延产 → 启动断更保护（X 短评 + 小红书旧文重发）。
  • 跨时区算错 → 排期日强制核对 ET ↔ BJT 偏移；X 可即时救场。
  • 国内大节假日 → 国内平台顺延 / 做节日版；海外照常。

04主理人优化建议与风险提示

优化建议

  1. 固定视觉 IP：1 个主色 + 1 套字体 + 真人出镜，9 端封面 / 头像统一，强化识别。
  2. 模板化降负担：小红书图文卡、视频号金句卡、X Thread 用固定模板，每周适配成本大降。
  3. 视频周前置生产：非视频周末拍剪，视频周只切片，平摊精力避免当周透支。
  4. evergreen 复用：小红书 / B站 / YouTube 旧文可周期性重发，吃长尾搜索红利。
  5. 数据驱动迭代：每周记录核心指标（公众号打开率、B站完播、YouTube CTR、X 互动），下轮调整选题与标题公式。

风险提示

  • ⚠️ 合规红线：微信禁诱导分享 / 外链裸链；小红书禁硬广 / 水印；YouTube / X / LinkedIn 含品牌合作须 FTC 披露（#ad）。
  • ⚠️ 版权：海外 BGM 用平台内置曲库（TikTok / IG）或 YouTube Audio Library；勿用国内平台音乐（海外无授权）；客户项目需脱敏。
  • ⚠️ 时区陷阱：每年 11 月–次年 3 月切换冬令时，海外北京时间点整体 +1h，排期日前务必核对。
  • ⚠️ 账号安全：抖音 / 小红书 / 视频号无完整自动 API，手动发布注意账号登录态与设备风控。
