# OmniGet：把 1,800+ 网站内容「下载 + 学习」揽进一个桌面

> 原文链接：https://appin.site/craft/omniget.html

---

## 一、它是什么

OMniGet 是一款**免费开源的桌面应用**（支持 Windows / macOS / Linux），干两件常被分开的事：把全网媒体**下载到本地**，并围绕这些媒体做**学习管理**。底层用 Rust + Tauri + Svelte 构建，协议是 GPL-3.0。

它和「纯下载器」（例如 yt-dlp 的图形壳）本质不同：后者只负责「拿到文件」，OmniGet 把**播放、阅读、笔记、复习**也做成了内置能力——文件落盘后，学习闭环直接在应用里走完，不用再跳去别的软件。

一句话定位：它是你的「个人媒体学习中枢」——把散落在各平台的课程、视频、播客、电子书，收拢到本地，并围绕它们建立「看 / 读 / 记 / 复习」的闭环。

## 二、核心能力拆解

把它的能力摊开看，核心是一条「下载 → 消费 → 学习」的链路：

- 全平台媒体下载

课程（Udemy / Hotmart / Kiwify / Teachable / Skool）、视频音频（YouTube / TikTok / Reddit / Instagram / Twitch / Vimeo / Bilibili）、亚洲平台（抖音 / 小红书 / 优酷 / 爱奇艺）、音乐（Spotify / SoundCloud / YouTube Music / Qobuz / Last.fm）、图库、以及种子 / 磁链——底层借力 yt-dlp 支持的 1,800+ 站点。

- 全局热键一键下载

按下 **Ctrl+Shift+D**（macOS 为 Cmd+Shift+D），直接抓取剪贴板里的链接开始下载，省去手动粘贴；B 站还支持 4K / HDR / 杜比深度下载。

- 内置课程播放器

带**时间戳笔记**、Pomodoro 计时器与进度看板，把「刷课」变成「学课」，知识点直接钉在时间轴上。

- PDF / EPUB 阅读器

专注模式阅读下载好的电子书，配合**笔记知识图**把零散笔记串成网络，而不是写完就忘。

- 音乐播放器

歌词、均衡器、流媒体连接一应俱全，下载的音频直接在本地整理收听，不必再开播放器。

- 插件系统

自带课程 / Telegram / 转换等插件，首次启动自装并**自更新**，也支持第三方扩展，能力可长可短。

## 三、支持的平台与服务

覆盖面是它最大的底气之一。按内容类型拆开看：

| 类别 | 代表平台 |
| --- | --- |
| 在线课程 | Udemy、Hotmart、Kiwify、Teachable、Skool |
| 视频 / 音频 | YouTube、Instagram、TikTok、Twitter/X、Reddit、Twitch、Vimeo、Bilibili |
| 亚洲平台 | 抖音、小红书、优酷、爱奇艺 |
| 音乐流媒体 | Spotify、SoundCloud、YouTube Music、Qobuz、Last.fm |
| 通用 | yt-dlp 支持的 1,800+ 站点、torrent / 磁链 |

此外还带字幕工坊、FFmpeg 转换、浏览器扩展，并支持 9 种语言、14 套主题，以及 portable.txt 便携模式（可放 U 盘带走）。

## 四、部署与安装

作为桌面应用，它没有「在线版」，需要装到自己机器上。各系统门槛不同：

- **Windows**：下载 .exe 便携版，解压即用，门槛最低。

- **macOS**：下载 .dmg 拖入应用，再运行一条 Terminal 命令解除 Gatekeeper 限制即可打开。

- **Linux**：`flatpak install wtf.tonho.omniget`，或用对应发行包安装。

- **开发构建**：克隆仓库后 `pnpm install` + `pnpm tauri dev`（需 Rust、Node 18+、pnpm）；生产构建 `pnpm tauri build`。

## 五、优势与局限

客观说，它的长板和短板都很清晰：

| 维度 | 优势 | 局限 |
| --- | --- | --- |
| 一体化 | 下载 + 播放 + 阅读 + 笔记 + 复习一条龙，文件全留本地 | 不是纯命令行工具，重活靠它自己的引擎，灵活性略低于脚本组合 |
| 平台覆盖 | 1,800+ 站点、含亚洲平台与音乐流媒体 | 登录态 / 付费课程需 Cookie 或授权，抓取范围受平台限制 |
| 技术栈 | Rust + Tauri 轻量跨平台，GPL-3.0 可自用可改 | 仅桌面端，无移动原生应用；插件 / 依赖初次配置仍需读文档 |
| 学习化 | 时间戳笔记 + 间隔重复 + Pomodoro，把「囤资源」变「真学习」 | 功能多，初次上手有学习曲线，建议先吃透下载 + 播放两条主线 |

## 六、怎么用 + 我的判断

典型路径是一条直线：**装好 → 复制链接 → 按热键下载 → 在播放器 / 阅读器里学 + 记笔记 → 间隔重复复习**。配置一次之后，后续基本靠「复制链接 + 热键」驱动。

我的判断：

- **适合**：囤课党 / 自学党、想把 YouTube / B 站课程 / 播客落地到本地系统学的人、注重隐私（数据全本地、不上云）的用户。

- **注意**：它是桌面应用，无移动端；登录态 / 付费内容取决于平台授权；功能多，先吃透「下载 + 播放」两条主线再扩展。

- **定位**：它不是「迅雷替代品」，而是「个人媒体学习中枢」——把分散在各平台的媒体收拢到本地，并围绕它们建立学习闭环。

它的价值不在「下得快」，而在让下载下来的内容，真正被「学掉」而非「堆着」。
