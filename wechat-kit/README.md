# 公众号适配包（微信公众号手动发布用）

本目录是为「手动发公众号」准备的成品包，配合已上线的 appin.site 技艺录文章使用。

## 目录结构
- `covers/` —— 微信封面图（**1080×460，比例 2.35:1**，已符合公众号首图规范）
  - `wechat-cover-aiworkskills.jpg`
  - `wechat-cover-daily-reading-2026-07-08.jpg`
- `md/` —— 公众号正文稿（Markdown，含标题/来源/正文，已去除站内导航）
  - `aiworkskills-wechat-ai-ops.md`
  - `daily-reading-2026-07-08.md`

## 使用方法
1. **封面图**：公众号后台「封面」→ 上传 `covers/` 里对应的 jpg 即可（比例已合规，无需再裁）。
2. **正文**：
   - 复制 `md/` 里某篇 `.md` 的全部内容；
   - 打开 Markdown 转公众号工具（如 **md2weixin.com**、135 编辑器、壹伴等）；
   - 粘贴后一键转换，再「复制到公众号」；
   - 在公众号后台微调样式 → 保存为草稿 / 定时发布。
   - 注：公众号原生后台**不支持** Markdown，必须经由转换工具。

## 说明
- 正文稿顶部已带「原文链接」，方便在公众号文末引导读者去 appin.site 看完整版/历史文章。
- 封面图由已上线文章的真实摄影头图本地裁剪合成，**未调用 AI 生图**（零积分消耗）。
- 如需改标题文字或换图，重新跑 `/tmp/gen_cover.py` 与 `/tmp/html2md.py` 即可。
