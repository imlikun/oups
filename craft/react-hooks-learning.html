React 学习笔记：从 Class 组件到 Hooks 的思维转变

刚开始学 React 时最困惑的就是函数式组件和 Hooks 的设计哲学。整理一下自己踩过的坑和最终理解。

Lkn 2026-06-03 约 300 字

为什么 Hooks 让我困惑

刚接触 React 的时候，教程用的还是 Class 组件。后来社区全面转向函数式组件 + Hooks，一下子很不适应。

最大的困惑是：没有 this 了，状态怎么管理？没有生命周期了，副作用怎么处理？

核心理解

useState——就是声明一个"会触发重渲染的变量"。 useEffect——就是"当某些值变化时，执行这个副作用"。 useRef——就是一个"跨渲染周期保持不变的盒子"。
