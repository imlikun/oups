一、你能白嫖到什么

Gemini 是 Google 出的 AI（和 ChatGPT 是一类东西）。它有个普通用户容易忽略的好处：能通过「API」免费调用——不用开会员、不用充钱。本文就教你怎么在国内把它配好，用任意支持「OpenAI 格式」的聊天客户端直接跟它对话。

免费额度有多少？ 免费层大约给到每天 500 次请求、每分钟 15 次。日常聊天完全用不完，只有疯狂刷屏才会超限，不用担心。

二、先准备这 3 样东西

1一个 Google 账号

没有就去注册一个，免费。

2一个能上境外网站的代理（梯子）

你平时能上 YouTube，一般就已经有了。后面关键是要让它「路由到 Google 的 API 域名」。

3一个聊天客户端

支持「自定义 API / OpenAI 兼容」设置的就行（比如你平时用的那个 AI 客户端）。本文不限定具体软件。

三、第 1 步：拿到 Google 的 Key

Key 就是一把「钥匙」，客户端靠它证明「我是合法用户」。获取步骤：

1. 打开 Google AI Studio：aistudio.google.com/apikey

2. 点「Create API Key」，会生成一个以 AI 或 AQ 开头的密钥。

3. 复制保存好。这串字符不要发给别人，泄露了就去后台点 Revoke 作废、重新生成。

四、第 2 步：确认网络能通 Google

这一步最容易踩坑，先看明白原理：

在浏览器打开 generativelanguage.googleapis.com 显示 404，不代表 API 不通。这个域名本来就没有网页首页，404 是正常的。

要判断 API 到底通不通，得访问完整路径（把 你的Key 换成你刚拿到的）：

https://generativelanguage.googleapis.com/v1beta/models?key=你的Key

如果返回一大串 JSON（模型列表），说明网络通了、Key 也有效。

关键坑：你的「梯子」必须能路由到 Google 的这个 API 域名。很多人 YouTube 能开、但 API 域名不通——因为有些节点只解锁了流媒体，没开放 Google Cloud。测试方法：把梯子切到「全局模式」，或选一个香港 / 新加坡 / 日本节点，再访问上面的地址。能返回 JSON，就说明这个节点能用。

五、第 3 步：在客户端填 4 个框

在客户端的「自定义 API / 模型设置」里，照着填这 4 项：

名称：随便写，如「Gemini」

Base URL（接口地址）：https://generativelanguage.googleapis.com/v1beta/openai

API Key：你刚拿到的 Google Key

模型名：gemini-3.1-flash-lite-preview

三个「不要」：① 模型名不要带 models/ 前缀；② 不要用 gemini-2.5-flash（新用户已不可用）；③ 不要用任何 Pro 系列（不免费，会报错）。

六、第 4 步：发一条消息测试

确保客户端走的是能访问 Google 的梯子节点（见第 2 步），然后发一句话，比如「你好」。

正常会收到回复——恭喜，已经免费用上 Gemini 了！

七、报错对照表（一眼看懂）

如果出错，先看错误码，对照下表：

错误码意思怎么办
401Key 错了或失效去 AI Studio 重新生成一个 Key
402余额不足（用了付费模型）换回 flash 系列，别用 Pro
403地区不允许换香港 / 新加坡 / 日本节点，或换其他模型
404路径或模型名写错核对第 5 步的 4 个框，尤其模型名
502网络到不了 Google梯子没路由到该域名，换节点或开全局

八、模型名千万别填错

✓能用（免费）

gemini-3.1-flash-lite-preview（推荐，免费额度最多）／ gemini-3.5-flash ／ gemini-3-flash

✗不能用

gemini-2.5-flash（新用户已下线）／ 任何 Pro 系列（不免费）／ 带 models/ 前缀的写法

九、小结

整件事就 4 步：

拿 Key→ 确认梯子通 Google→ 客户端填 4 个框→ 发消息测试

核心坑只有两个：① 梯子要能路由到 Google 的 API 域名（不是只开 YouTube）；② 模型名填 gemini-3.1-flash-lite-preview，别用 2.5 和 Pro。搞定后，就是免费、国内可用的 Gemini。
