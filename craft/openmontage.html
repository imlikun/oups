一、它是什么

OpenMontage 是一个基于 AI Agent 架构的开源视频制作系统。它包含 12 条生产流水线、52 个专业工具、500+ 智能体技能——从创意构思、脚本撰写、素材生成到剪辑合成，全流程由 AI 自动编排。一句话：你用自然语言描述想要什么视频，AI Agent 帮你完成从零到成品的所有工作。

它和「提示词 → 5 秒视频」类工具的根本区别在于：普通 AI 视频工具只产出几秒片段，没有调研、脚本、剪辑、质控的完整链路；OpenMontage 是端到端的流水线，且这些技能协同模拟真实视频团队的工作流——把代码 / Codex 直接转化为完整视频内容。

一句话定位：你的 AI 编程助手（Claude Code / Cursor / Copilot / Windsurf / Codex）就是编排器，OpenMontage 让它自主跑完「调研 → 提案 → 脚本 → 场景 → 素材 → 剪辑 → 渲染」全链路。

二、核心能力拆解

OpenMontage 的能力可以拆成三层：流水线定义「做什么样的视频」，工具提供「怎么生产」，技能教 AI「怎么像个专家一样用工具」。

  1. 12 条生产管线

    动画解说、电影预告片、纪录片蒙太奇、Talking Head、短视频批量生产、播客转视频、本地化 / 配音、屏幕演示等——覆盖视频制作全场景。

  2. 52 个专业工具

    视频生成、图像生成、TTS 配音、音乐生成、音频混音、字幕、画面增强、分析等，拼出完整制作链路。

  3. 500+ 智能体技能

    每个工具都有对应的技能文件，教 AI 如何像专家一样使用它们——这是它区别于「裸调 API」的关键。

  4. 参考视频驱动

    粘贴一个你喜欢的 YouTube / Reel / TikTok 链接，AI 分析它的节奏、风格、结构，帮你创建差异化版本，而非凭空猜。

三、真实素材 · 零成本

它不依赖「付费用 AI 生成几秒片段」，而是尽量用真实、可商用的素材把视频做出来：

  • 真实素材纪录片：支持从 Archive.org、NASA、Wikimedia Commons 等免费开放资源中检索和剪辑真实影像，做真正的纪录片蒙太奇。
  • 零 API Key 也能跑：内置 Piper TTS（离线语音合成）、免费素材源、Remotion / HyperFrames 合成引擎，完全免费也能制作视频。
  • CLIP 语义检索 + 免费素材库：把「想要什么画面」翻译成可检索的真实素材，而不是硬生成。

和普通 AI 视频工具的区别

Sora / Kling 类只生成几秒片段，没有调研、脚本、剪辑、质控的完整链路；OpenMontage 是端到端流水线，且 Agent-First——你的 AI 编程助手就是编排器。

四、成本治理与质量把关

把 AI 编程助手变成视频工作室，最怕两件事：账单炸了、成品是废品。OpenMontage 用两套机制兜住：

维度做法
成本治理7 维度评分 + 成本预估 + 预算上限 + 人工审批，杜绝意外账单；每次制作前先给预估。
质量把关渲染前验证、渲染后自检（ffprobe + 帧采样 + 音频分析 + 字幕检查），确保不出废品。
决策可追溯每个 Provider 选择跨 7 维度打分，留可追溯决策日志；每个创作决策需你审批。

五、部署与安装

前置依赖（门槛不高，但要齐）：

  • Python 3.10+（python.org）
  • Node.js 18+（nodejs.org）
  • FFmpeg 最新版（brew install ffmpeg / apt install ffmpeg）
  • 一个 AI 编程助手：Claude Code / Cursor / Copilot / Windsurf / Codex
# 克隆仓库 git clone https://github.com/calesthio/OpenMontage.git cd OpenMontage # 一键安装 make setup # 运行演示 make demo # 渲染特定演示 ./render-demo.sh world-in-numbers # Windows：若 npm install 失败 npx --yes npm install # 配置（把 .env.example 复制为 .env 并填 Key） cp .env.example .env

然后在你的 AI 编程助手里打开项目，直接说一句话就行：「做一个 60 秒的动画解说，讲神经网络怎么学习」，或真实影像路线：「做一个 75 秒的纪录片蒙太奇，关于雨中的城市生活，只用真实素材、无旁白、挽歌基调、配乐」。
