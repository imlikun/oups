一句话说明

TrendRadar 就像一个「AI 私人编辑」——它每天帮你盯着微博、抖音、知乎等 35+ 个平台的热搜，只把你关心的内容筛选出来，一键推到你的微信/钉钉/飞书上。你不用再一个个 App 去刷，打开聊天窗口就能看完当天最重要的新闻。

这是什么？

想象你订了一份「私人报纸」：

  • ❌ 传统方式：每天买 35 份报纸，每份翻一遍，信息量爆炸但真正想看的只有 3 条
  • ✅ TrendRadar：一个编辑帮你盯着这 35 份报纸，只剪下你感兴趣的那几条，装进信封送到你桌上

核心能力

1. 全网热点聚合

默认监控 11 个平台：微博、抖音、知乎、B站热搜、今日头条、百度热搜、华尔街见闻、财联社、澎湃新闻、凤凰网、贴吧。基于 newsnow 数据源，可扩展到 35+ 个平台。

2. 三种智能推送模式

模式 工作方式 适合谁
daily（当日汇总） 推送当天所有匹配新闻 管理者/普通人
current（当前榜单） 推送当前榜单上的匹配新闻 自媒体/创作者
incremental（增量监控） 只推新增的，绝不重复 投资者/交易员

3. 精准关键词筛选

6 种筛选语法：

  • 普通词：基础匹配，包含任一即捕获
  • +词汇：必须词，必须同时包含
  • !词汇：过滤词，排除干扰
  • @数字：数量限制
  • /pattern/：正则表达式
  • =文本：备注显示名称

4. AI 分析简报（v5.0.0+）

让 AI 先读完所有新闻，生成一份深度分析报告，包含：

  1. 核心热点态势——今天最热门的几个话题
  2. 舆论风向争议——有争议的观点和讨论
  3. 异动与弱信号——还没上热搜但值得关注的信息
  4. 研判策略建议——基于以上分析的行动建议

支持 100+ AI 提供商（DeepSeek、OpenAI、Gemini、Claude、Ollama 本地等）。

5. MCP 对话分析（v3.0.0+）

基于 MCP 协议，可以在 Claude Desktop、Cherry Studio 等 AI 客户端里直接和 TrendRadar 对话分析，支持 21+ 种智能分析工具。

安装部署

方案一：GitHub Actions（30秒部署）

  1. Fork 仓库
  2. 配置 Secrets（WEWORK_WEBHOOK_URL 等）
  3. 手动运行一次 workflow 测试
  4. ⚠️ 每 7 天需要「签到」一次续期

方案二：Docker 部署（推荐）

# 1. 克隆项目
git clone https://github.com/sansan0/TrendRadar.git
cd TrendRadar

# 2. 配置推送渠道和关键词
nano docker/.env
nano config/frequency_words.txt

# 3. 启动
docker compose up -d

# 4. 查看状态
docker exec -it trendradar python manage.py status

推送渠道

支持 9+ 渠道：企业微信、个人微信、飞书、钉钉、Telegram、邮件、ntfy、Bark、Slack。

企业微信配置最简单：进入群聊 → 消息推送 → 添加机器人 → 复制 Webhook 地址 → 配置到 WEWORK_WEBHOOK_URL。
