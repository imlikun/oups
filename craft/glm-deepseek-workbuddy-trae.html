智谱 GLM + DeepSeek 接入 WorkBuddy / Trae：国内直连，免费用上大模型

两个国产大模型，都兼容 OpenAI 接口，国内直连、不用任何额外工具，免费额度够日常使用。本文手把手把它们接进 WorkBuddy 和 Trae，零基础也能照做。

两个都不需要任何网络工具，注册即用，是新手入门最稳的选择。

前置准备：各自拿一个 Key

  • 智谱：打开 open.bigmodel.cn，注册后在「API Keys」页面新建密钥，复制 sk- 开头的 Key
  • DeepSeek：打开 platform.deepseek.com，注册后在「API Keys」页面新建密钥，复制 sk- 开头的 Key

⚠️ Key 只显示一次，存到安全的地方；泄露了就在控制台删除重开。

一、智谱 GLM 接入

1. 接入 WorkBuddy

  1. 打开 WorkBuddy，点左下角 设置 ⚙️
  2. 进入 模型 设置页 → 自定义模型 → 添加模型
  3. 提供商选 其他 / Custom（⚠️ 不要选「智谱官方」，那个会跳腾讯云收费通道）
  4. 填三个参数：
字段填写内容
API Base URLhttps://open.bigmodel.cn/api/paas/v4
API Key你的智谱 Key（sk- 开头）
模型名称glm-4-flash（免费）

想要更强可换 glm-4.7-flash 或 glm-5.1（按量计费）。

保存，模型选择器里选中 glm-4-flash 即可对话，调用走智谱免费额度。

2. 接入 Trae

  1. Trae → 设置 → AI Providers → 选 Custom 或 OpenAI Compatible
  2. 填：
字段填写内容
API URLhttps://open.bigmodel.cn/api/paas/v4/chat/completions
API Key你的智谱 Key
模型名称glm-4-flash

保存设为默认，写代码就能用 GLM 补全和对话。

3. 验证一下（Python）

from openai import OpenAI

client = OpenAI(
    api_key="你的智谱Key",
    base_url="https://open.bigmodel.cn/api/paas/v4"
)

r = client.chat.completions.create(
    model="glm-4-flash",
    messages=[{"role": "user", "content": "用一句话介绍智谱 GLM"}]
)
print(r.choices[0].message.content)

4. 免费额度

glm-4-flash 永久免费；新用户另有 2000 万 token 体验额度，轻量长期使用足够。

二、DeepSeek 接入

1. 接入 WorkBuddy

  1. 打开 WorkBuddy → 设置 → 模型 → 自定义模型 → 添加模型
  2. 提供商选 其他 / Custom
  3. 填：
字段填写内容
API Base URLhttps://api.deepseek.com/v1
API Key你的 DeepSeek Key（sk- 开头）
模型名称deepseek-chat

需要深度推理换 deepseek-reasoner。

保存，选中即可对话。

2. 接入 Trae

  1. Trae → 设置 → AI Providers → Custom / OpenAI Compatible
  2. 填：
字段填写内容
API URLhttps://api.deepseek.com/v1/chat/completions
API Key你的 DeepSeek Key
模型名称deepseek-chat

保存设为默认。

3. 验证一下（curl）

curl https://api.deepseek.com/v1/chat/completions \
  -H "Authorization: Bearer 你的DeepSeekKey" \
  -H "Content-Type: application/json" \
  -d '{"model":"deepseek-chat","messages":[{"role":"user","content":"用一句话介绍 DeepSeek"}]}'

4. 免费额度

网页端 / App 完全免费；API 新用户赠 100 万 token 体验，之后按量计费、价格很低。

常见问题

  • 要装代理 / 加速工具吗？ 不需要。两个都是国内直连，正常网络环境直接通。
  • Trae 报错 Invalid BaseURL？ API URL 必须填完整路径（含 /v1/chat/completions），不能只填域名。
  • 模型名大小写？ 严格照填 glm-4-flash / deepseek-chat，错一个字符都会报模型不存在。
  • 能两个都加吗？ 可以。WorkBuddy / Trae 都支持添加多个自定义模型，按需切换。

总结
