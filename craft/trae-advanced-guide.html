文章摘要

把两篇高赞 Trae 实测教程整合为一站式攻略：第一部分用 Agnes AI 等 5 款免费 API 给 Trae Work 接上独立通道，不排队、不限量、国内直连；第二部分彻底讲透 Trae 的三类记忆与导入导出，让换电脑、改项目、组队协作都不丢 AI 习惯。所有接口参数已核对至 2026 年 7 月最新实测。

5 款免费 API 实测参数 Agnes 三模态永久免费 8n+1 视频帧规则 三类记忆区分 .trae 软链接迁移

为什么给 Trae 接第三方免费 API

经常用 Trae Work 的朋友应该都有体会，官方模型的痛点越来越明显：高峰期排队动辄几百上千人，写代码、调逻辑突然卡住；官方额度从日限改成周限，重度开发、批量调试根本不够用；自带模型品类太少，复杂推理、长文档、图文创作能力都比较单一；官方付费模型价格也不便宜。

好在 Trae Work 原生支持 OpenAI 标准自定义模型——不用破解、不用装插件，简单几步接入第三方免费 API，就能解锁不用排队、永久免费、多模型随意切换的满血体验，完全可以替代官方模型。

一、最优解：Agnes AI 三模态永久免费（2026.7 实测）

经过逐一实测，Agnes AI 是目前唯一国内可直连、文本 + 画图 + 短视频全模态永久免费的 API，没有任何套路，个人开发者可以零成本无脑用。

1. 免费规则（永久有效，无套路）

  • 注册超简单：普通国内邮箱就能注册，不需要境外手机号、不用绑卡、全程零充值。
  • 免费模型全覆盖：agnes-2.0-flash（文本/编码）、agnes-image-2.1-flash（AI 生图）、agnes-video-v2.0（短视频）全部免费开放。
  • 额度超级宽松：无月度调用上限、无 Token 总量限制，只有超高并发会轻微限流，日常个人开发完全感知不到。
  • 商用边界清晰：个人学习、项目测试、低并发小型商用随便用；企业高并发业务，建议联系官方开通不限速专属通道。

2. 手把手接入 Trae Work（三步复制即用）

  1. 第一步：获取专属 API Key

    打开官方开发者平台 platform.agnes-ai.com，用邮箱注册登录，左侧菜单「设置」→「API 密钥」→「创建新的密钥」。重点提醒：密钥只显示一次，前缀为 sk-agnes-xxxx，一定要及时复制备份。

    实测有效全局接口地址：https://apihub.agnes-ai.com/v1

  2. 第二步：Trae 自定义模型配置

    打开 Trae Work → 设置 → 模型 → 添加模型，拉到最底部选「自定义配置」，按下面精准填写：

    配置项精准填写内容（2026.7 实测）
    API 格式OpenAI Chat Completions 格式
    自定义请求地址https://apihub.agnes-ai.com/v1
    模型 IDagnes-2.0-flash
    API 密钥个人 sk-agnes-xxxx 密钥
    高级配置默认无需修改
  3. 第三步：接入验证（必做）

    把模型切换为刚添加的 agnes-2.0-flash，在对话框输入 你是什么大模型？，只要正常返回模型信息，就说明接入成功，可以直接用来写代码、调逻辑、改文案。

二、进阶：用 Skill 接入生图 / 生视频

这里有个关键细节：新版 Trae 只支持文本模型直连，画图、生成短视频必须通过 Skill 技能封装才能调用，这也是目前唯一能稳定用的方法：

  1. 在 Trae 对话框输入固定指令：我想要使用 Agnes Image 2.1 模型生图、Agnes Video 2.0 模型生成视频，访问它的 API 平台 https://agnes-ai.com/doc/overview，并将它打包成一份 Skill。
  2. 等待自动生成 agnes-ai 技能文件和 skill.md，把文件里所有 YOUR_API_KEY 替换成你自己的密钥。
  3. 回到 Trae 技能界面，上传修改好的文件，就能免费无脑用 AI 画图、生成短视频。
避坑重点

视频生成有个硬性规则：帧数必须是 8n+1（比如 81、121、161 帧），填错必报错；另外高峰期服务器拥挤会出现 500 报错，不用慌，错开高峰重试即可。图片和视频模型严禁直接当对话模型接入，必须用 Skill 方式。

如果你也用 WorkBuddy（腾讯的桌面级 AI 助手），接入方式差不多，而且搭配起来更香——WorkBuddy 负责操控本地文件，Agnes 负责免费算力。模型下拉菜单点「+ 添加自定义模型」，提供商选「自定义 / Custom」，参数填：

配置项填写内容

API 密钥你的 sk-agnes-xxxx
模型名称（文本）agnes-2.0-flash

接好后可以一键批量解析 PDF、汇总 Excel、给代码批量加注释、自动生成封面，完全不消耗 WorkBuddy 积分。注意：用文件操控前，先在设置里把目标文件夹权限打开。

五、接入报错排查（FAQ）

  • 模型无响应 / 接不通：大概率是 BaseURL 填错、密钥带了空格换行、或密钥过期封禁。
  • 弹出 429 限流：所有免费模型都有每分钟请求上限，批量任务拆分分次执行，间隔 1-2 秒重试即可恢复。
  • 图文 / 视频 500 报错：多模态模型不能直接自定义接入，必须用 Skill 技能模式；视频严格遵守 8n+1 帧数规则，错峰使用更稳定。
  • DeepSeek 调用失败：新用户官方直连额度有限，切换硅基流动聚合通道，稳定性翻倍。
  • 频繁卡顿、断连：日常开发优先用硅基流动、智谱；需要极速响应就切 Groq，避开早晚高峰。
  • WorkBuddy 依旧扣积分：第三方 API 算力绝对不扣积分，只有软件本地原生操作会少量消耗，属正常规则。

七、Trae 记忆体系：三类记忆先分清

很多人用 Trae 还会踩另一类坑：C 盘被记忆缓存占满、项目改名记忆丢失、换电脑记忆无法迁移。先把记忆类型分清，后面所有修改、导出、导入才不会出错——三类记忆完全独立：

记忆类型作用范围存储位置可迁移性
全局智能记忆所有项目通用（个人编码习惯、通用要求）系统 AppData 固定目录支持整体迁移、复制备份
项目专属记忆仅当前项目（架构、代码规范、业务规则）项目根目录 /.trae 隐藏文件夹禁止改名移动，仅可复制复用
会话聊天记忆单条对话上下文，临时任务记录内置会话数据库仅可导出，无法一键导入复原
硬核结论

Trae 官方没有一键修改记忆目录的可视化开关，两类记忆修改规则不同，不可混为一谈！

八、存储目录与修改（C 盘爆满怎么办）

项目记忆 .trae：千万别动

项目记忆默认在 ./.trae/ 隐藏文件夹。绝对不要重命名、剪切、移动、删除它——一旦项目路径变动，Trae 会判定为新项目，原有的项目记忆、规则、技能全部清零。正确做法是直接编辑内部记忆文本来修改 AI 项目约束规则。

全局记忆：软链接迁移改盘

全局记忆默认路径：Windows %APPDATA%\Trae CN、macOS ~/Library/Application Support/Trae CN、Linux ~/.config/Trae CN。如果 C 盘爆满，可以用软链接把它整体迁移到 D/E 盘，软件无感、升级不失效：

# Windows（管理员 CMD，先彻底退出 Trae） mklink /D "%APPDATA%\Trae CN" "D:\TraeData\Trae CN" # macOS / Linux ln -s /Volumes/Data/TraeData/TraeCN ~/Library/Application\ Support/Trae\ CN

注意：不能单独拆分迁移记忆子文件夹，只能整体迁移 Trae CN 根目录；不想迁移的话，用 Ctrl+Shift+P 执行 Trae: Clear Cache 清缓存不删记忆。

九、导入导出全套（换电脑 / 协作 / 备份）

  • 项目长期记忆（最重要）：核心文件是 AGENTS.md + 根目录 /.trae 文件夹。导出直接复制这两个打包；导入时粘贴到新项目根目录，重启 Trae 自动加载（模式一：全盘复用），或只复制 project_rules.md / user_rules.md 文本（模式二：选择性复用）。
  • 全局个人记忆：直接复制全局根记忆文件夹（Windows C:\Users\用户名\.trae），新电脑粘贴覆盖即一键复原。
  • AI 自动零散记忆：Ctrl+Shift+P 输入 Trae: Open Memory Panel，逐条复制面板内记忆文本保存。
  • 会话聊天记录：对话右上角「…」→ 导出 → 生成 md 文件；无法一键导入，新建空白会话粘贴内容即可让 AI 接续上下文。

十一、高阶技巧：把记忆变成团队资产

  • 团队协作记忆共享：把项目内 .trae 文件夹和 AGENTS.md 提交到 Git 仓库，团队拉取代码后 Trae 自动同步统一的编码、业务记忆规范。
  • 自制通用记忆模板：写好通用代码风格、注释格式、交付要求存进 AGENTS.md，每次新建项目直接导入，无需重复告知 AI 编码习惯。
  • 记忆减负技巧：不要堆砌冗余记忆，定期清空记忆面板无效条目；业务变更直接修改 rules 文档，比删除记忆更稳定。

十二、总结与重要须知

把两件事做对，Trae 的体验会完全不一样：第一，接上 Agnes AI 等免费 API，排队和额度焦虑一次性解决；第二，理清三类记忆 + 做好 .trae / AGENTS.md 的导入导出，换电脑、改项目、组队都不丢 AI 习惯。截止 2026 年 7 月，这套方案已经非常成熟——不用花钱、不用排队、额度充足，写代码、改文案、做 AI 创作全场景覆盖。

  • 所有免费 API 适合个人学习、项目测试、低并发小型商用；企业高并发业务建议升级付费通道。
  • 永久免费不代表无限制轰炸调用，高频刷屏容易触发临时限流，正常开发使用完全没问题。
  • AI 画图、生视频严格遵守内容合规规则，违规内容会直接封禁密钥且无法恢复。
  • 免费平台政策会动态微调，正式项目上线前建议简单核对一遍官方最新规则。
