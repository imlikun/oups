一、这是什么？

yt-dlp 是基于 Python 的开源命令行音视频下载工具，是已经停更的 youtube-dl 的活跃分支（fork）。

简单来说：你在 B 站、YouTube、Twitter、TikTok、抖音等任何视频网站看到一个视频，想下载到本地永久保存——yt-dlp 复制粘贴链接就能下。

二、它解决什么痛点？

日常下载视频有这几个老问题：

  1. 网页右键另存为只能下到 m3u8/分片，合并不了
  2. 需要特定分辨率或仅音频，网页端不提供选择
  3. 批量下载播放列表，网页一个一个点太慢
  4. 下载被中断要从头开始，网络环境差的情况
  5. 广告平台加水印，需要原始清晰度
  6. 想看视频但不想登录网站，要个本地播放器就能看

yt-dlp 一次解决：

  • 一行命令下载任意视频——支持 1800+ 网站
  • 完整格式选择——从 144P 到 8K，从纯音频到 HDR
  • 播放列表批量——一次下一个专辑/频道/合集
  • 断点续传——中断后再跑自动接上
  • 原始质量——不经过网站转码，画质无损
  • 离线观看——下完本机随便看，不依赖网站

五、怎么安装？

方式一：单文件二进制（推荐）

# macOS
sudo curl -L https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp_macos -o /usr/local/bin/yt-dlp && sudo chmod a+rx /usr/local/bin/yt-dlp

# Linux
sudo curl -L https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp -o /usr/local/bin/yt-dlp && sudo chmod a+rx /usr/local/bin/yt-dlp

方式二：pip

pip install -U yt-dlp

方式三：Homebrew

brew install yt-dlp
brew install ffmpeg  # 必备依赖

⚠️ 必备依赖

ffmpeg 是音视频合并和转码的必需品，不装会出现"下载的视频没声音"问题：

brew install ffmpeg        # macOS
sudo apt install ffmpeg    # Linux
choco install ffmpeg        # Windows

六、5 分钟上手

  1. 基础下载（最佳质量） 
    yt-dlp -P ~/Downloads/yt-dlp "VIDEO_URL"
  2. YouTube 带 Cookie（避坑）
    YouTube 现在会屏蔽未认证请求，必须带浏览器 Cookie： 
    yt-dlp --cookies-from-browser chrome "VIDEO_URL"
  3. 只下音频（提取 MP3） 
    yt-dlp -x --audio-format mp3 "VIDEO_URL"
  4. 下载字幕 
    yt-dlp --write-subs --sub-langs zh-Hans,en "VIDEO_URL"
  5. 指定分辨率 
    # 720P
yt-dlp -f "bestvideo[height<=720]+bestaudio/best[height<=720]" "VIDEO_URL"
# 1080P
yt-dlp -f "bestvideo[height<=1080]+bestaudio/best[height<=1080]" "VIDEO_URL"
# 4K 优先
yt-dlp -f "bestvideo[height>=2160]+bestaudio/best" "VIDEO_URL"
  6. 下载播放列表 
    # 整张播放列表
yt-dlp -o "%(playlist)s/%(playlist_index)02d - %(title)s.%(ext)s" "PLAYLIST_URL"
# 只下前 5 集
yt-dlp -I 1:5 "PLAYLIST_URL"
  7. 下载前查看可用格式 
    yt-dlp -F "VIDEO_URL"

七、进阶玩法

并发分片下载（速度翻倍）

yt-dlp -N 8 "VIDEO_URL"

智能格式选择

yt-dlp -S "res:1080,ext:mp4:m4a" "VIDEO_URL"

按章节切片

yt-dlp --split-chapters -o "chapter:%(title)s/%(section_title)s.%(ext)s" "VIDEO_URL"

自动跳过广告（SponsorBlock）

yt-dlp --sponsorblock-remove sponsor "VIDEO_URL"
yt-dlp --sponsorblock-remove all "VIDEO_URL"  # 移除所有非必要片段

录制直播

yt-dlp --live-from-start "LIVE_STREAM_URL"

限速下载

yt-dlp --limit-rate 5M "VIDEO_URL"

    十三、配置文件

    把常用选项放到 ~/.config/yt-dlp/config：

    # ~/.config/yt-dlp/config
--no-mtime
--write-subs
--sub-langs zh-Hans,en
--embed-metadata
--embed-thumbnail
--cookies-from-browser chrome
-P ~/Downloads/yt-dlp
-o "%(uploader)s/%(upload_date>%Y-%m-%d)s - %(title)s.%(ext)s"

    配置后直接 yt-dlp "URL" 即可。

    十五、总结

    yt-dlp 的核心价值：

    • 不是替代浏览器下载，而是替代一整套商业下载软件——1800+ 网站支持
    • 不是技术宅专属，而是内容创作者的瑞士军刀——日常 3 个命令就够
    • 不是黑科技，而是合规开源——Unlicense 协议，可商用可改造
    • 不是装完就完，而是持续进化——每日构建永远在跟平台反爬升级
    如果你经常需要保存视频到本地（学习资料、创作素材、备份存档），yt-dlp 是 2026 年唯一正确的选择，没有之一。
