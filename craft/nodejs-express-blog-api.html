Node.js 实战：用 Express 搭建一个个人博客 API

前端工程师学后端，Express 是最平滑的切入点。从零搭建一个支持文章 CRUD 的 REST API，包括路由设计、中间件、数据持久化。

李坤 2026-04-15 约 705 字

为什么选 Express

  • 学习曲线平缓，和前端 JavaScript 语法一致
  • 生态成熟，中间件非常丰富
  • 部署简单，一台 VPS 就能跑

项目结构

my-blog-api/
├── server.js          # 入口文件
├── routes/            # 路由定义
│   ├── articles.js
│   └── uploads.js
├── middleware/        # 中间件
│   └── auth.js
└── data/             # JSON 文件数据库
    └── articles.json
