MultiPost 是一个开源免费的浏览器扩展，由 leaperone 团队开发维护。支持知乎、微博、小红书、抖音、B站等 20+ 国内平台，以及 YouTube、Bluesky 等国际平台。

一、这是什么？

MultiPost 是一个开源免费的浏览器扩展，核心功能就一句话：写一次内容，一键发布到所有社交平台。告别复制粘贴，告别多平台来回切换。

四、怎么用？

方式一：浏览器扩展（最简单）

Chrome 用户：Chrome Web Store；Edge 用户：Edge Add-ons。安装后在任意平台打开编辑器，写好内容，选择目标平台，一键发布。

方式二：在线编辑器

访问 md.multipost.app，在线编辑 Markdown 内容并一键发布到多平台。

方式三：开发者集成

// 通过扩展 API 发布 const result = await chrome.runtime.sendMessage({ action: 'publish', platforms: ['weibo', 'zhihu', 'xiaohongshu'], content: { title: '文章标题', body: '文章正文...', images: ['image1.jpg', 'image2.jpg'] } });

六、优缺点

✅ 优点

  • 真正免费 — 无需付费、无需注册、无需 API Key
  • 平台覆盖广 — 27 个处理器，主流平台基本全覆盖
  • 零门槛 — 浏览器装个扩展就能用
  • 开源透明 — 代码完全开放，可以自己改
  • 活跃维护 — 2026 年 6 月仍在密集更新
  • 开发者友好 — 提供 RESTful API 和扩展 API

⚠️ 注意事项

  • 各平台的 DOM 结构可能因更新而变化，需要持续维护适配
  • 某些平台可能有反自动化机制
  • 大量平台同时发布时建议分批操作，避免触发风控
  • 视频类平台（抖音/B站/视频号）可能需要额外配置

七、一句话总结

如果你需要在 3 个以上平台发相同或类似的内容，MultiPost 能帮你省掉大量"复制→切换→粘贴→发布"的重复劳动。免费开源，装上就用。
